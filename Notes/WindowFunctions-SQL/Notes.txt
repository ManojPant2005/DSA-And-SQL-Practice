1. What is a Window Function?

A Window Function performs calculations across a set of rows related to the current row without grouping the result into a single row.

Difference Between GROUP BY and Window Function
GROUP BY
SELECT DepartmentId, AVG(Salary)
FROM Employees
GROUP BY DepartmentId;


| Function      | Purpose                  |
| ------------- | ------------------------ |
| ROW_NUMBER()  | Unique row numbering     |
| RANK()        | Ranking with gaps        |
| DENSE_RANK()  | Ranking without gaps     |
| NTILE()       | Divide into groups       |
| LEAD()        | Next row value           |
| LAG()         | Previous row value       |
| FIRST_VALUE() | First value in partition |
| LAST_VALUE()  | Last value in partition  |
| SUM() OVER()  | Running total            |
| AVG() OVER()  | Average per partition    |
| MIN() OVER()  | Minimum value            |
| MAX() OVER()  | Maximum value            |
